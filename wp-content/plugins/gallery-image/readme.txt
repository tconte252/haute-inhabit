== Description ==

Create unlimited images galleries.

For more information, check out <a href="http://www.danycode.com/gallery-image/">Gallery Image</a>

Plugin Features:

- Create unlimited images galleries
- Support for PNG, JPG, GIF
- Add your galleries into posts or pages
- Easy to use

== Installation ==

1. Upload the Gallery Image plugin to your blog and activate it. 
2. Go to the Option page and set the gallery width and the vertical margin.
3. Go to the Add Gallery page, set a name for the gallery and add images for the gallery ( to have a great result in a single gallery use images with the same width and height ) , click on the save button to complete the submission.
4. Go to the Manage Galleries page and pick up the code under the Embed Code column.
5. Go in a post or in a page and paste the Embed Code inside the HTML of your post or page.
6. Check your post or page to see the result.
